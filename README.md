# MyFirstProjectInTrae

This is a Python project created in Trae.

## Description

This project is a simple Python application that prints "Hello from myfirstprojectintrae!".

## Dependencies

The project uses the following dependencies:

- langchain
- python-dotenv
- langchain-community

## Getting Started

To get started with this project, you will need to have Python installed on your system.

### Prerequisites

You can install the dependencies using pip:

```bash
pip install -r requirements.txt
```

### Running the application

To run the application, execute the following command:

```bash
python main.py
```
